Пределы функций

Определение предела

Число A называется пределом функции f(x) при x стремящемся к a, если для любого ε > 0 существует δ > 0 такое, что из |x - a| < δ следует |f(x) - A| < ε.

Основные свойства пределов

  • Предел суммы равен сумме пределов
  • Предел произведения равен произведению пределов
  • Предел частного равен частному пределов (если предел знаменателя не равен нулю)

Примеры вычисления пределов

lim(x→0) sin(x)/x = 1

lim(x→∞) (1 + 1/x)x = e
