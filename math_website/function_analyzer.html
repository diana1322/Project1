Анализ функций

Квадратичная функция

Рассмотрим функцию вида f(x) = ax2 + bx + c

Где параметры могут быть заданы в URL: a=2, b=3, c=1

Влияние параметров на график

  • Параметр a: определяет направление ветвей параболы (a > 0 - вверх, a < 0 - вниз)
  • Параметр b: влияет на смещение вершины параболы по горизонтали
  • Параметр c: определяет точку пересечения с осью Y

Исследование функции

Для полного исследования функции нужно найти:

  1. Область определения
  2. Точки пересечения с осями
  3. Экстремумы (используя производную)
  4. Интервалы возрастания и убывания
  5. Точки перегиба

Примеры ссылок с параметрами:

  • f(x) = x²
  • f(x) = 2x² + 4x + 1
  • f(x) = -x² + 2x + 3
